## 简介

大二暑期在深圳实习，上班时间偷偷学习MEAN架构（MongoDB, Express, AngularJS, Node.js），写了个简单的聊天系统。

Github：[github.com/HugHoo/Chat](github.com/HugHoo/Chat)

主要技术
- Node.js
- AngularJS
- Express

CSS framework
- [Materialize CSS](http://materializecss.com/)

Third-party lib
- [Socket.io](https://socket.io/)

```

cd Chat

npm install

node app.js
```
